# 📚 SIPUS - Sistem Informasi Perpustakaan

Sistem manajemen perpustakaan modern untuk mengelola buku, anggota, dan peminjaman secara efisien.

## ✨ Fitur

### 🔐 Autentikasi
- **3 Role** — Admin, Petugas, Anggota
- **Login Aman** — autentikasi dengan role-based access
- **Demo Akun** — tersedia akun demo untuk testing

### 📊 Dashboard
- **Statistik** — jumlah buku, anggota, peminjaman
- **Grafik** — visualisasi data peminjaman
- **Overview** — ringkasan aktivitas perpustakaan

### 📚 Manajemen Buku
- **CRUD Buku** — tambah, edit, hapus buku
- **Filter Kategori** — filter berdasarkan kategori
- **Pencarian** — cari buku berdasarkan judul/penulis
- **Detail Buku** — informasi lengkap buku

### 👥 Manajemen Anggota
- **CRUD Anggota** — tambah, edit, hapus anggota
- **Data Lengkap** — nama, alamat, telepon, dll
- **Status Anggota** — aktif/nonaktif

### 🔄 Peminjaman & Pengembalian
- **Proses Peminjaman** — catat peminjaman buku
- **Proses Pengembalian** — catat pengembalian
- **Status** — tracking status peminjaman
- **Denda** — hitung denda keterlambatan

### 🏷️ Kategori
- **Manajemen Kategori** — CRUD kategori buku
- **Filter** — filter buku berdasarkan kategori

### 📈 Laporan
- **Laporan Peminjaman** — daftar peminjaman
- **Statistik** — grafik dan statistik
- **Export** — export laporan

### ⚙️ Pengaturan
- **Konfigurasi** — pengaturan aplikasi
- **Profil** — edit profil pengguna

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- [Tailwind CSS](https://tailwindcss.com/) — utility-first CSS
- [Alpine.js](https://alpinejs.dev/) — reactive framework
- [Chart.js](https://www.chartjs.org/) — grafik
- [Font Awesome](https://fontawesome.com/) — icons
